How the Magic Works: Algorithm Breakdown

1. Background Capture

The core function first captures a static frame of the scene without the target object . This `self.background` image is crucial for the replacement step. It auto-updates every 3 seconds for dynamic environments.

// Capture initial background...
`self.background = np.flip(frame, axis=1)`

2. Color Segmentation (HSV)

The live camera frame is converted to the HSV color space for robust color detection. A mask is created using user-defined (or preset) HSV ranges (e.g., Red or Green) to isolate the cloaking fabric.

`hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)`
`mask = cv2.inRange(hsv, lower_bound, upper_bound)`

3. Mask Refining & Blending

The raw mask is cleaned using morphological operations (`cv2.morphologyEx`) to remove noise. The final image is created by blending the background where the mask is present and the live frame where the mask is absent.

`res1 = cv2.bitwise_and(bg, bg, mask=mask)`
`final = cv2.addWeighted(res1, 1, res2, 1, 0)`
